# 分类体系参考

本文档提供多种分类标签体系，用于为仓库分组添加有趣的标签，增强记忆和识别度。

## 使用说明

在生成 REPO-GROUPS.md 时，请选择一个分类体系。标签会出现在分组标题的注释中：

```markdown
## Go-Practice <!-- 标签 -->
```

每个分类体系包含多个标签，使用数字序号选择标签，体验更流畅。

## 5. 🎮 自定义体系

**特点**：可以基于任何你喜欢的主题，完全自定义标签。

**建议主题**：
- 游戏角色（如：马里奥、林克、索尼克、劳拉）
- 动漫角色（如：路飞、鸣人、一护、悟空）
- 历史人物（如：诸葛亮、张良、韩信、项羽）
- 星座（如：白羊、金牛、双子、巨蟹）
- 元素周期表（如：氢、氦、锂、铍）
- 颜色（如：红、橙、黄、绿、蓝）
- 方向（如：东、南、西、北、中）
- 等等...

**使用方式**：
选择此体系后，可以为每个分组自由定义标签名称。

## 使用建议

1. **选择符合兴趣的体系**：这样能更好地记忆和识别
2. **保持一致性**：一个配置文件建议使用同一个分类体系
3. **标签唯一性**：尽量让每个分组使用不同的标签
4. **标签关联性**：尽量让标签与分组特点有所关联
5. **可重入性**：可以随时修改或更换标签

## 标签选择方式

在增强 Prompt 中，所有标签都使用数字序号选择：
- 输入 1-10 的数字选择对应的标签
- 直接回车使用推荐标签（标注 ⭐）
- 无需输入标签名称，体验更流畅
